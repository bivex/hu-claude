# Примеры запуска команд специалистов (Agent Teams)

## Утренний блок «Тело и разум»
```
Создай команду из 3 специалистов для утренней программы:
- Инструктор по медитации: проведи утреннюю медитацию 15 мин
- Инструктор йоги: составь утреннюю последовательность 20 мин
- Нутрициолог: предложи оптимальный завтрак

Пусть каждый специалист работает независимо, затем синтезируй единую утреннюю программу.
```

## Глубокая проработка проблемы
```
Создай команду для проработки [проблемы] с 4 специалистами:
- Психолог: анализ эмоционального компонента
- Психосоматолог: анализ телесного компонента
- НЛП-практик: работа с убеждениями
- Лайф-коуч: план действий

Пусть они обсудят находки друг с другом и создадут интегрированный план.
```

## Карьерный sprint
```
Создай команду для карьерного прорыва:
- Карьерный консультант: аудит текущей ситуации
- Экзекьютив-коуч: стратегия развития
- Бизнес-тренер: ролевая игра собеседования
- Специалист по профориентации: анализ сильных сторон

Задача: создать 90-дневный карьерный план.
```

## Еженедельный обзор
```
Создай команду для воскресного ревью:
- Координатор: подведение итогов недели
- Лайф-коуч: рефлексия и инсайты
- Менатор: стратегический взгляд
- Таролог: расклад на новую неделю
```

## Антистресс-сессия
```
Создай команду для экстренной помощи при стрессе:
- Медитация: дыхательная практика
- Психолог: техника заземления
- Гипнолог: быстрая релаксация

Работайте последовательно: сначала дыхание, потом заземление, потом релаксация.
```
